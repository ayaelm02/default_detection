Welcome to DDDDDPDC's documentation!
===================================

**DDDDDPDC** (/DDDDDPDC/) is a Python library for default detection in curved pieces.
It tells you what kind of default present in the piece or wether its good or not machined at all from an image you insert <http://localhost:8501/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::
DDDDDPDC has its documentation hosted on Read the Docs.
   This project is under active development.
